Adresy používané na internetu

Na internetu se používají 3 základní adresy.

Adresa zdroje (URL, Uniform Resource Locator). Používá se pro přesnou identifikaci dokumentů na internetu, více známá jako webová adresa.

Identifikátor zdroje (URI, Uniform Resource Identifier). Je to obecnější koncept než URL, kterému je nadřazený.

A identifikátor síťového rozhraní na internetu IP adresa. Číslo, které jednoznačně identifikuje síťové rozhraní v počítačové síti, která používá IP protokol. V současné době je nejrozšířenější IPv4, která používá 32bitové IP adresy. Jsou zapisovány dekadicky po jednotlivých oktetech (tj. po osmicích bitů), například 192.168.0.2 .

Seznam významných termínů a zkratek

HTML
Značkovacího jazyk používaný pro tvorbu webových stránek.
HTTP
Internetový protokol určený pro komunikaci s WWW servery.
Validátor
Validátor je počítačový program používaný ke kontrole platnosti nebo syntaktické správnosti fragmentu kódu nebo dokumentu.
Server
Server je v informatice obecné označení pro počítač, který poskytuje nějaké služby, nebo počítačový program, který tyto služby realizuje.
